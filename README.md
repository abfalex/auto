# auto
Сайт с объявлениями о продаже автомобилей разных марок.
Данные парсятся с сайта `drom.ru`.

# Как запустить оффлайн сайт в браузере?
 Для запуска сайта вам понадобится Python третьей версии.

Скачайте код с GitHub. Затем установите необходимые библиотеки командой:

```sh
pip install -r requirements.txt
```
Запустите сайт при помощи команды:
```sh
python render_website.py
```
Версия локального сайта будет доступен по [ссылке](http://127.0.0.1:5500) - `http://127.0.0.1:5500`.
